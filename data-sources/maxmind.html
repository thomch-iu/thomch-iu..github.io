Maxmind DB


Overview

  • A database that maps an IP address to a Geolocation.
  • For this project the companies' Country geolocation data is used.
  • Data used by security researchers and marketers alike.
  • Used the free GeoLite2 Country dataset
  • Downloaded binary Country database for analysis
  • Downloaded csv Country database for display in presentation
  • Same data in both databases

Collection Strategy:

  • Accessed data through company's GeoLite2 dataset download page

Summary Statistics:

  • Records Collected:
    • Roughly 250 countries / governmental areas
    • Roughly 37,000 IPv4 blocks
    • Roughly 26,000 IPv6 blocks
  • Coverage Dates: April 2022
